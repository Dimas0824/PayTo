## How to Become a Contributor

Kami menerima kontribusi dalam berbagai bentuk:

- 💻 **Code contributions** - Bug fixes, features, refactoring
- 📖 **Documentation** - Improving docs, tutorials, examples
- 🐛 **Bug reports** - Finding and reporting bugs
- 💡 **Feature ideas** - Suggesting new features
- 🎨 **Design** - UI/UX improvements, mockups
- 🧪 **Testing** - Writing tests, QA
- 🌍 **Translations** - Translating docs and UI
- 📣 **Community** - Helping others, answering questions

Lihat [CONTRIBUTING.md](CONTRIBUTING.md) untuk panduan lengkap.
